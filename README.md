# flask_blog

Required python3

## Install Required Packages
### MacOs/Linux
```
# virutal environment
  virtualenv -p python3 env     
# activate 
  source env/bin/activate
# install requirement
  pip install -r requirements.txt
```
### Windows
```
# virutal environment
  python3 -m virtualenv env     
# activate 
  env\Scripts\activate.bat
# install requirement
  pip install -r requirements.txt
```
## Run app
```
  python run.py
```
